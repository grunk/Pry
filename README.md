# Pry Framework
===

Another PHP 5 Framework. Require at least PHP 5.3
  
 [API](http://pry.oroger.fr/api/)
 +
 [Documentation & Example](http://pry.oroger.fr/doc/)

# Installation
Via composer : 

```
"repositories": [
    {
        "type": "vcs",
        "url": "https://github.com/grunk/Pry"
    }
],
"require": {
    "pry/pry" : "@dev"
}```
